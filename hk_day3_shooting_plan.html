拍摄计划

按行程执行的拍摄时间线

每个地点都用“动作 → 环境 → 细节 → 离开”来拍，避免只留下漂亮空镜。

0109:30

从铜锣湾出发 → 西九龙

这段做开场转场。用港铁、扶梯、站台和海边第一眼，把观众带到 M+。

🎬 必拍镜头

  • 酒店出门 / 电梯 / 街头早晨。
  • 港铁站台、扶梯、人流。
  • 西九龙海边第一眼。
  • 人物走向 M+，留建筑空间。

✍️ 字幕感觉

早上从铜锣湾出发，去西九龙看展。
来香港，我最期待的是花一个上午慢慢看展。
0210:00

M+ 博物馆

M+ 是这一天的审美主场。拍建筑、空间、人和作品的关系，不要只拍展品。

🎬 必拍镜头

  • M+ 建筑外观：竖屏 + 横屏各一条。
  • 入口、票、扶梯、指示牌。
  • 展厅大空间：人站在画面里。
  • 看展背影：让人和作品产生关系。
  • 纪念品店：书、海报、明信片、手部动作。
  • 西九龙海边 / 天光作为转场。

✍️ 字幕感觉

在很大的空间里，人会变得很安静。
我喜欢看展时那种不需要说话的时间。

⚠️ 注意 / 避免

  • 不要怼拍展品细节过多；重点是空间和人的状态。
  • 注意馆内规定，不适合拍的展品就不要拍。
0312:30

旺角 / 太子午餐

这里是从艺术馆转入旧街区的过渡。不要只拍吃什么，拍人流、旧楼、茶餐厅门口和城市密度。

🎬 必拍镜头

  • 旺角 / 太子街道和斑马线。
  • 茶餐厅 / 面店门口。
  • 菜单、餐具、热气、手拿杯子。
  • 吃完从店里走出来。
  • 旧楼、楼梯、门牌。

✍️ 字幕感觉

从西九龙出来，香港突然变得很密。
午餐不只是吃饭，是进入旧街区的入口。
0413:30

Infree Records

这是全程最适合你账号气质的点之一。重点拍旧楼、推门、翻唱片、灯光、窗外和坐下听歌。

🎬 必拍镜头

  • 上楼过程：楼梯、门牌、走廊。
  • 推门进店，第一视角慢一点。
  • 手翻唱片，拍 2–3 条不同角度。
  • 黑胶封面、唱片架、灯光。
  • 坐下听歌 / 拿起杯子 / 看窗外。
  • 离开时回头拍店门。

✍️ 字幕感觉

旧楼里有一家很安静的唱片店。
在这里，时间像被调慢了一点。

⚠️ 注意 / 避免

  • 不要拍成店铺测评；它是情绪场景，不是探店说明书。
0515:00

前深水埗配水库 / 深水埗街区

如果配水库预约顺利，可以拍空间结构；如果时间紧，就把重点放在深水埗街区纹理。

🎬 必拍镜头

  • 配水库外部 / 入口 / 结构线条。
  • 深水埗街区：鸭寮街、桂林街、布料、旧招牌。
  • 唐楼窗户、晾衣、电线、手写价牌。
  • 人穿过摊位 / 等红绿灯 / 买小吃。
  • 街边小吃热气和手拿食物。

✍️ 字幕感觉

深水埗不是精致的香港，但很有生活的纹理。
这里的美不在远处，在手写价牌和旧楼窗户里。

⚠️ 注意 / 避免

  • 不要只拍“脏乱旧”，要拍质感、层次和人的生活。
  • 街区人多，注意不要冒犯性怼脸拍。
0617:15

志莲净苑 / 南莲园池

行程里志莲净苑有 16:30 关门风险，所以不要把它作为唯一核心。能提前到就拍志莲外部和安静氛围；赶不上就把南莲园池作为黄昏结尾。

🎬 必拍镜头

  • 如果 16:00 前到：拍志莲外部、木构、路径、人物背影。
  • 南莲园池：金亭、倒影、水面、石路、树。
  • 人物站在中景，不要站太满。
  • 黄昏暖光到蓝调的变化。
  • 离开时夜色和港铁转场。

✍️ 字幕感觉

这一天最后，想在一个安静的地方等天黑。
从旧街走到园林，香港突然慢了下来。

⚠️ 注意 / 避免

  • 志莲净苑殿内禁拍，请尊重规定。
  • 如果时间赶不上，不要硬拍；南莲园池足够做结尾。
07晚上

回铜锣湾 / 酒吧

这段只做收尾和备用素材。不要让酒吧破坏 Day 3 的安静主线，除非氛围真的很好。

🎬 必拍镜头

  • 港铁回程窗外 / 站台。
  • 铜锣湾夜晚第二次出现，作为呼应。
  • 酒吧门口或一杯饮品特写。
  • 回酒店路上背影。

✍️ 字幕感觉

走了一整天，最后又回到铜锣湾。
这座城市很吵，但有些角落很安静。
